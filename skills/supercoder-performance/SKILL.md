# Supercoder Performance

The Performance specialist optimizes system speed and efficiency.

## Responsibilities

### Performance Analysis
- Profile applications
- Identify bottlenecks
- Measure metrics
- Set baselines

### Optimization
- Code optimization
- Database tuning
- Caching strategies
- CDN configuration

### Monitoring
- Performance metrics
- Load testing
- Capacity planning
- Trend analysis

## Performance Process

### 1. Profiling
1. Identify slow operations
2. Profile code execution
3. Analyze database queries
4. Measure network calls

### 2. Optimization
1. Prioritize improvements
2. Optimize critical path
3. Add caching
4. Tune database

### 3. Verification
1. Measure improvement
2. Load test
3. Monitor in production
4. Set up alerts

## Optimization Areas

### Frontend
- Bundle size
- Image optimization
- Lazy loading
- CDN usage

### Backend
- Query optimization
- Caching
- Connection pooling
- Async processing

### Database
- Index optimization
- Query analysis
- Partitioning
- Sharding

## Tools

- **Profiling**: Chrome DevTools, py-spy, Node clinic
- **Load Testing**: k6, Locust, JMeter
- **APM**: New Relic, Datadog, OpenTelemetry
- **Database**: EXPLAIN, Query analyzer

## Best Practices

- **Measure first** - Don't guess
- **Optimize critical path** - Focus on bottlenecks
- **Cache wisely** - Balance freshness
- **Monitor production** - Real-world performance
