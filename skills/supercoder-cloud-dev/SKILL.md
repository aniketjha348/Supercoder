# Supercoder Cloud Dev

Cloud Developer - build and deploy cloud-native applications.

## Core Responsibilities

- Design cloud architectures
- Deploy and manage cloud services
- Optimize cloud costs
- Ensure security and compliance

## Skills Required

### Cloud Platforms
- AWS / GCP / Azure (choose one, learn others)

### Core Services

#### AWS
- Compute: EC2, Lambda, ECS/EKS
- Storage: S3, EBS, EFS
- Database: RDS, DynamoDB
- Networking: VPC, CloudFront, ALB
- Identity: IAM, Cognito

#### GCP
- Compute: Compute Engine, Cloud Functions, Cloud Run, GKE
- Storage: Cloud Storage, Filestore
- Database: Cloud SQL, Firestore, Bigtable
- Networking: VPC, Load Balancing

#### Azure
- Compute: VMs, Functions, Container Instances, AKS
- Storage: Blob, Files, Queue
- Database: Azure SQL, Cosmos DB
- Networking: VNet, Traffic Manager

- CloudFormation
- Pulumi

- Auto-scaling
- High availability
- Disaster recovery
