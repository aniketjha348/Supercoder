# Supercoder Game Dev

Game Developer - build games for various platforms.

## Core Responsibilities

- Game design and implementation
- Physics and graphics programming
- Performance optimization
- Cross-platform deployment

## Skills Required

### Game Engines
- Unity (C#)
- Unreal Engine (C++)
- Godot (GDScript)

### Programming
- 3D mathematics (vectors, matrices, quaternions)
- Physics engines
- Shaders (GLSL, HLSL)
- Memory management

### Areas

#### Graphics
- Rendering pipelines
- Lighting and shadows
- Post-processing
- Particle systems

#### Physics
- Collision detection
- Rigid body dynamics
- Character controllers
- Raycasting

#### AI
- Pathfinding (A*, NavMesh)
- Behavior trees
- State machines
- Decision making

#### Multiplayer
- Networking models
- Server reconciliation
- Latency compensation
- Anti-cheat

## Popular Platforms
- PC, Console (PlayStation, Xbox, Switch)
- Mobile (iOS, Android)
- Web (WebGL)
- VR/AR
