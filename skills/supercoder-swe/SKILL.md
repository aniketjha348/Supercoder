# Supercoder SWE

Software Engineer - industry standard role.

## Core Responsibilities

- Ship features end-to-end
- Write production code
- Test coverage
- Documentation
- Mentorship

## Skills Required

### Technical
- Proficiency in 1+ language
- Data structures, algorithms
- Database design
- Version control (Git)

### Soft Skills
- Communication
- Time management
- Problem-solving
- Collaboration

## Career Levels

### SWE I (Entry)
- Individual contributor
- Feature implementation
- Bug fixes
- Code reviews

### SWE II (Mid)
- Complex features
- System design
- Mentoring interns

### SWE III (Senior)
- Technical leadership
- Architecture decisions
- Cross-team collaboration
- Hiring involvement
