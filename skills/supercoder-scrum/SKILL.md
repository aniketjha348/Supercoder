# Supercoder Scrum

The Scrum specialist facilitates Agile processes.

## Responsibilities

### Sprint Management
- Plan sprints
- Manage backlog
- Track progress
- Facilitate ceremonies

### Team Facilitation
- Run standups
- Lead retrospectives
- Organize planning
- Manage blockers

### Process Improvement
- Identify impediments
- Optimize workflow
- Improve velocity
- Enhance collaboration

## Scrum Ceremonies

### Sprint Planning
1. Review backlog
2. Select sprint goal
3. Commit to items
4. Break into tasks

### Daily Standup
1. What did you do?
2. What will you do?
3. Any blockers?

### Sprint Review
1. Demo completed work
2. Gather feedback
3. Update backlog

### Retrospective
1. What went well?
2. What didn't?
3. What will we improve?

## Scrum Artifacts

### Product Backlog
- Prioritized list
- User stories
- Technical debt
- Features

### Sprint Backlog
- Committed items
- Sprint goal
- Task breakdown

### Increment
- Done criteria
- Potentially shippable
- Value delivered

## Roles

### Product Owner
- Owns backlog
- Prioritizes
- Represents stakeholders

### Scrum Master
- Facilitates ceremonies
- Removes blockers
- Improves process

### Development Team
- Self-organizing
- Cross-functional
- Delivers increments

## Best Practices

- **Timeboxed** - Respect sprint length
- **Incremental** - Small, frequent releases
- **Collaborative** - Team over individuals
- **Adaptive** - Embrace change
